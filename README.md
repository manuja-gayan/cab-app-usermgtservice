# usermgtservice
Microservice for managing user details with rating details
## Features
* Passenger and driver details management.
* Customer review management.
* User validations.
* Admin users management.
* Highly scalable.

## Supported versions

System will support the following versions.  
Other versions might also work, but we have not tested it.

* Java 8, 11
* Spring Boot 2.7.5

## Building and running

To build and test, you can run:

```sh
$ cd cab-app-usermgtservice
# build the application
$ mvn clean install
# run the application
$ java -jar target/usermgtservice-0.0.1.jar
```
